# Changelog

## en-US

## 0.1.0

- Initial Power BI Business Intelligence plugin.
- Added core Power BI reporting, modelling, DAX, Power Query, and delivery skills.
- Added expert-reduction operating model skills.
- Added industrial process intelligence and source-system-aware adapters.
- Added ERP/CRM and enterprise source-product coverage.
- Added QM/QMS/EHS and WHS/WMS source packs.
- Added process-chain KPI visibility for additional industrial KPI chains.
- Added AI/KI orchestration, RAG, guardrails, hallucination checks, insight confidence, and autonomous BI sprint skills.
- Added bilingual `en-US` and `de-DE` documentation standard.
