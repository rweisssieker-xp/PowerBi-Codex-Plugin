# Privacy Policy

## en-US

This local plugin stores no secrets and does not call external services by itself. Generated artifacts may reference customer data, credentials, or connection strings only when the user provides them. Keep credentials in local environment variables, Power BI Desktop credential storage, gateways, or approved secret stores.

## de-DE

Dieses lokale Plugin speichert keine Secrets und ruft von sich aus keine externen Services auf. Generierte Artefakte duerfen Kundendaten, Credentials oder Connection Strings nur referenzieren, wenn der Nutzer sie bereitstellt. Credentials gehoeren in lokale Umgebungsvariablen, Power BI Desktop Credential Storage, Gateways oder freigegebene Secret Stores.
