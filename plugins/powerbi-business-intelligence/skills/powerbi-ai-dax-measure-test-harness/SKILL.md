# Power BI AI DAX Measure Test Harness

Use this skill to make DAX expert replacement verifiable.

## Test Areas

- KPI contract to DAX measure mapping.
- Expected value tests for known slices.
- Reconciliation totals against source systems.
- Date-basis tests for order, invoice, delivery, posting, or fiscal dates.
- Filter context tests for customer, product, plant, warehouse, region, and owner.
- Regression tests against previous measure outputs.

## Output Requirements

- Include DAX query, expected value, tolerance, test data scope, owner, and pass/fail criteria.
- Mark measures without test evidence as certification risks.
