# Power BI AI Agent Workforce Orchestrator

Use this skill for multi-role BI delivery.

## Agent Roles

- Consultant Agent, Source Mapper, Semantic Modeler, DAX Expert, Power Query Expert, Designer, QA Agent, CoE Reviewer, Documentation Owner, and Release Manager.
- Handoffs, required inputs, required outputs, quality gates, and evidence.
- Escalation when evidence, source access, or owner approval is missing.

## Output Requirements

- Produce orchestration plan, agent sequence, handoff artifacts, gates, and final release package.
- Do not skip QA or owner acceptance in autonomous flows.
