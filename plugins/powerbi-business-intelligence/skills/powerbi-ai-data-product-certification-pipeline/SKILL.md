# Power BI AI Data Product Certification Pipeline

Use this skill to certify reusable Power BI data products.

## Pipeline Gates

- KPI contracts and source reconciliation.
- Semantic model graph and naming standards.
- DAX tests and measure documentation.
- Power Query load contracts and refresh evidence.
- RLS/OLS, sensitivity labels, sharing, and export controls.
- Report UX review, owner acceptance, release evidence, and CoE decision.

## Output Requirements

- Produce gate status, blockers, remediation backlog, certification decision, and release recommendation.
- Certification cannot pass with missing owner or missing test evidence.
