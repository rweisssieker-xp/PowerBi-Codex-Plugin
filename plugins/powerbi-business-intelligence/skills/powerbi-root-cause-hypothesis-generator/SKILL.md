# Power BI Root Cause Hypothesis Generator

Use this skill when the dashboard must explain why a KPI moved.

## Hypothesis Pattern

1. Define symptom, time window, comparison baseline, and affected scope.
2. Generate candidate hypotheses from process knowledge and source-system events.
3. Identify evidence required to test each hypothesis.
4. Rank hypotheses by plausibility, impact, and confidence.
5. Produce next analysis and data gaps.

## Output Requirements

- Include hypothesis, process step, source evidence, test measure, confidence, and action owner.
- Keep hypotheses separate from proven root causes.
