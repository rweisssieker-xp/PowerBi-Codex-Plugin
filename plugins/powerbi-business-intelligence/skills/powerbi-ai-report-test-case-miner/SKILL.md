# Power BI AI Report Test Case Miner

Turn real historical edge cases into reusable UAT and regression tests.

## Output Contract

- Source events, edge-case class, affected KPI/report/model, and historical evidence.
- Test case with filters, expected result, tolerance, source trace, owner, and severity.
- Regression pack placement and retest cadence.
- Coverage gaps and new synthetic demo-data needs.

## Checks

- Anonymize or avoid sensitive case details in reusable packs.
