# Power BI Data Quality Issue Router

Use this skill when data issues block reporting, certification, or trust.

## Issue Types

- Missing keys, duplicate keys, orphan facts.
- Invalid dates, currencies, units of measure.
- Missing required attributes.
- Conflicting master data across systems.
- Late postings, deleted records, archival gaps.
- Schema drift or API extraction failure.

## Output Requirements

- Include issue, affected KPI/report, source system, owner, severity, evidence, remediation, SLA, and retest.
- Route ownership based on source of record, not the report builder.
