# Power BI AI Model Explainability for Business

Use this skill to explain Power BI outputs to non-technical stakeholders.

## Explanation Areas

- Why a number has a certain value.
- Which filters, slicers, relationships, and date basis affect it.
- Which source tables and transformations feed it.
- Which DAX measure logic applies.
- Which data is missing, stale, or excluded.

## Output Requirements

- Produce business-readable explanation, technical trace, assumptions, and confidence.
- Avoid exposing sensitive implementation details to unauthorized audiences.
