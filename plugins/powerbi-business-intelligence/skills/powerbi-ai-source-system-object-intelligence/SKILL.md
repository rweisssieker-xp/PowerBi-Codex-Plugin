# Power BI AI Source-System Object Intelligence

Use this skill for deep source-system mapping.

## Object Intelligence Areas

- SAP, Dynamics, Salesforce, Oracle, ServiceNow, MES, WMS, QMS, EPM, and Fabric objects.
- Table/API candidates, joins, keys, document flow, status lifecycle, and delta mechanisms.
- Typical data quality issues and missing history risks.
- Validation against real metadata before implementation.

## Output Requirements

- Produce object map, join candidates, source assumptions, validation plan, and risk list.
- Treat vendor-memory mappings as candidates until metadata confirms them.
