# Power BI Auto Report Layout Generator

Use this skill after the business question and model grain are known.

## Layout Pattern

- Executive overview: KPI cards, trend, variance, action list.
- Driver analysis: waterfall, decomposition, Pareto, scatter.
- Flow/cohort/process: funnel, retention, process timeline, event variants.
- Exceptions: prioritized table with owner, impact, reason, next action.
- Detail and reconciliation: drillthrough, transaction detail, source totals.

## Output Requirements

- Include page purpose, visuals, fields/measures, slicers, drillthrough, bookmarks, and validation notes.
- Avoid visuals whose grain conflicts with the semantic model.
