# Power BI Cost and Capacity Optimizer

Use this skill when reducing operational cost is part of reducing Power BI expert load.

## Analysis Areas

- Capacity: CPU, memory, throttling, refresh concurrency, visual query load.
- Storage: dataset/model size, duplicate imports, unused columns, incremental refresh, retention.
- Workspaces: inactive reports, duplicate semantic models, abandoned dataflows, owner gaps.
- Architecture: import vs DirectQuery vs Direct Lake, aggregation tables, certified datasets.
- Licensing: Pro, PPU, Premium, Fabric capacity, external sharing needs.

## Output Requirements

- Produce savings opportunities with risk, owner, effort, expected benefit, and verification metric.
- Do not recommend deletion without owner approval and retention review.
