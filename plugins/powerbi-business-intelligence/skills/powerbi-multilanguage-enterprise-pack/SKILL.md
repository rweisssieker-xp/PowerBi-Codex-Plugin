# Power BI Multilanguage Enterprise Pack

Use this skill for global industrial companies.

Default language pair for this plugin: `en-US` and `de-DE`.

## Coverage

- KPI names and descriptions.
- Report titles, page titles, slicers, tooltips.
- Business glossary.
- Acceptance documents.
- Training material.
- Data quality messages.
- Owner and action labels.

## Output Requirements

- Include source language, target language, approved terminology, non-translatable system terms, and owner review.
- Preserve source-system object names unless a business alias is explicitly approved.
