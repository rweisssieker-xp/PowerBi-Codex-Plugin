# Power BI Prompt to Backlog

Use this skill when Power BI work must enter an engineering or analytics delivery backlog.

## Backlog Outputs

- Epic: business outcome and delivery scope.
- User stories: audience, decision, report/model capability.
- Data tasks: source access, extraction, mapping, validation.
- Model tasks: facts, dimensions, relationships, measures, RLS.
- Report tasks: pages, visuals, drillthrough, filters.
- QA tasks: reconciliation, performance, RLS, UAT.
- Definition of done: quality gate, documentation, acceptance.

## Output Requirements

- Include dependencies and blockers.
- Write acceptance criteria in testable language.
