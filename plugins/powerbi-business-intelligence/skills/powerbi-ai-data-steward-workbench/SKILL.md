# Power BI AI Data Steward Workbench

Create an operating workbench for BI data stewardship.

## Output Contract

- Data-quality issue inventory, affected KPIs, source objects, severity, financial impact, and owner candidate.
- Root-cause grouping, correction backlog, SLA, escalation, and retest criteria.
- Steward dashboard, workflow fields, and value tracking.
- Recurrence analysis and prevention recommendation.

## Checks

- Distinguish source correction from reporting workaround.
