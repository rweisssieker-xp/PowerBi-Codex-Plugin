# Power BI AI Migration Autopilot

Use this skill for migration to governed Power BI.

## Migration Areas

- Source artifact inventory, owner, usage, business value, and technical risk.
- KPI and measure extraction, semantic model target, and report redesign.
- Data source replacement, native connector design, and refresh modernization.
- Consolidation, retirement, certification, and release sequence.

## Output Requirements

- Produce migration plan, target architecture, effort, risks, tests, and decommission checklist.
- Separate lift-and-shift from redesign and consolidation work.
