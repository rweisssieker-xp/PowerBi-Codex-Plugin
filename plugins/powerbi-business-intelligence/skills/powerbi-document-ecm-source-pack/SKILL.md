# Power BI Document ECM Source Pack

## Vendors

SharePoint, OpenText, DocuWare, M-Files, Hyland OnBase, Box, Google Drive, Microsoft Syntex.

## Source Objects

- Document, folder, library, metadata, version, approval, retention label, contract, invoice, quality record, attachment, access log.

## Power BI Patterns

- Document backlog.
- Contract metadata.
- Invoice processing status.
- Quality document compliance.
- Retention and approval status.

## Output Requirements

- Include document ID, version, metadata quality, owner, retention, access constraints, and source link strategy.
