# Power BI AI Expert Replacement Scorecard

Use this skill to quantify expert replacement.

## Score Areas

- Consultant work replaced: intake, scope, backlog, solution framing.
- Data analyst work replaced: questions, insights, root-cause, narrative.
- Designer work replaced: pages, visuals, drilldowns, UX review.
- Modeler work replaced: model graph, facts, dimensions, relationships.
- DAX/M work replaced: measures, queries, tests, optimization.
- QA/CoE work replaced: quality gates, certification, evidence.
- Documentation work replaced: bilingual docs, release notes, handover.

## Output Requirements

- Produce score by expert role, evidence, limitations, remaining human approvals, and estimated effort saved.
- Do not count unvalidated generated content as replaced expert work.
