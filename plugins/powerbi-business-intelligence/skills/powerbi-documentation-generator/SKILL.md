# Power BI Documentation Generator

Use this skill whenever a report, model, or delivery pack needs handover-ready documentation.

**Required companion skill:** Use `powerbi-bilingual-documentation` for all substantial documentation outputs.

## Documentation Outputs

- Report documentation: pages, visuals, filters, drillthrough, audience, decisions.
- Semantic model documentation: tables, relationships, measures, hierarchies, hidden fields.
- KPI catalog: definitions, DAX, owner, validation source, certification status.
- Data flow: source systems, connector, transformations, refresh, gateway.
- Security matrix: roles, RLS/OLS rules, test users, sensitivity labels.
- Operations: owner, refresh SLA, monitoring, support, known risks.

## Output Requirements

- Separate confirmed facts from assumptions.
- Include update owner and review cadence.
- Provide matching `en-US` and `de-DE` documentation sections or paired files.
