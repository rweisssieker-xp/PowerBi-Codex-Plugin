# Power BI Visual Layout Spec Generator

Convert report intent into a build-ready visual layout specification.

## Output Contract

- Pages, audience, primary decision, viewport assumptions, and navigation.
- Visual inventory with type, purpose, fields, measures, filters, interactions, drillthroughs, tooltips, and sort order.
- Layout density, page hierarchy, accessibility, conditional formatting, and exception states.
- Validation checklist for readability, misleading visuals, mobile constraints, and owner acceptance.

## Checks

- Use visual choice based on decision task.
- Keep exception/action tables close to KPI drivers.
