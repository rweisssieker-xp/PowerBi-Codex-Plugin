# Power BI Automated Source Metadata Scan

Define repeatable metadata scanning for BI sources.

## Output Contract

- Scan scope, schedule, source systems, permissions, owners, and safe authentication notes.
- Objects, fields, relationships, measures, refresh settings, lineage, labels, and usage to collect.
- Change detection, schema drift, data contract impact, and alert routing.
- Scan output tables and documentation refresh trigger.

## Checks

- Never store credentials.
- Respect source-system rate limits and admin approval.
