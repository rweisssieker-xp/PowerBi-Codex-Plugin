# Power BI Agent Workforce

Use this skill when a request normally needs several Power BI experts.

## Virtual Roles

- BI Architect: scope, model pattern, ownership, delivery path.
- Data Source Engineer: connector, extraction, folding, gateway, refresh.
- Semantic Modeler: facts, dimensions, relationships, RLS/OLS.
- DAX Expert: measure contracts, context, performance, validation.
- Report UX Designer: page flow, visuals, drillthrough, exception workflow.
- Fabric Architect: Import, DirectQuery, Direct Lake, Lakehouse, Warehouse, capacity.
- QA Reviewer: validation, reconciliation, acceptance checks.
- CoE Lead: standards, certification, portfolio, guardrails, adoption.

## Output Requirements

- Assign roles to the request, list each role's decision, and merge them into one action plan.
- Avoid duplicating work between roles; each role should own a distinct concern.
