# Power BI SAP Fabric Migration Playbooks

Plan migrations with parity, governance, and decommissioning evidence.

## Migration Patterns

- SAP BW to Fabric/Power BI.
- SAP extract to Lakehouse/Warehouse.
- Excel/Access to governed semantic model.
- Tableau/Qlik/SSRS to Power BI.
- Legacy warehouse marts to certified data products.

## Output Contract

- Source inventory, target architecture, migration waves, parity tests, KPI mapping, security mapping, and cutover plan.
- Rebuild vs reuse decision.
- Decommission checklist and owner acceptance.
