# Power BI AI Refactoring Agent

Use this skill to improve existing Power BI artifacts.

## Refactoring Areas

- Duplicate or unused measures, columns, tables, and visuals.
- Wrong relationships, ambiguous paths, many-to-many risks, and weak model grain.
- DAX performance, measure naming, display folders, and calculation patterns.
- Power Query folding, refresh reliability, type conversion, and schema drift.
- Report layout, visual overload, missing drilldowns, and inaccessible UX.
- Missing owner, documentation, tests, and certification evidence.

## Output Requirements

- Produce refactor backlog with priority, risk, affected artifacts, test plan, and expected benefit.
- Separate safe cleanup from breaking migration work.
