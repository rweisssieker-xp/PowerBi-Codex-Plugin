# Power BI EPM Planning Source Pack

## Vendors

SAP Analytics Cloud Planning, SAP BPC, Anaplan, Board, OneStream, Oracle EPM, Workday Adaptive Planning, Jedox, Pigment, CCH Tagetik, Planful.

## Source Objects

- Scenario/version, plan, forecast, budget, actual bridge.
- Account, cost center, profit center, entity, product, customer, currency, fiscal period.
- Driver-based assumptions, allocation rules, consolidation adjustments, commentary.

- Plan/actual/forecast variance.
- Rolling forecast.
- Price/volume/mix.
- Working capital scenarios.
- Management commentary.

## Output Requirements

- Include scenario/version semantics, fiscal calendar, currency, writeback owner, approval status, and reconciliation to actuals.
