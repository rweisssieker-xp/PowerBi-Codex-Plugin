# Power BI Automated DAX Test Generation

Generate repeatable tests for DAX measures.

## Output Contract

- Measure, KPI contract, source total, expected values, filters, and contexts.
- Test cases for totals, blanks, zero denominators, time intelligence, segments, RLS, and edge cases.
- Execution pattern, evidence capture, owner, and pass/fail rules.
- Regression pack placement.

## Checks

- Include source reconciliation for certified measures.
