# Power BI Process DAX Pattern Library

Design DAX patterns from KPI contracts and process grain.

## Pattern Families

- Aging and SLA breach.
- Funnel conversion and stage leakage.
- Backlog and WIP.
- OTIF and service level.
- Inventory coverage, blocked stock, and obsolescence.
- Margin bridge, price-volume-mix, cost variance.
- Cohorts, churn, retention, and renewal.
- Exception counts, financial exposure, and action closure.

## Output Contract

- KPI contract before formula.
- Required columns, relationships, date roles, and filter behavior.
- Measure catalog with base, derived, display, and diagnostic measures.
- Test cases for totals, blanks, edge cases, RLS, and reconciliation.
