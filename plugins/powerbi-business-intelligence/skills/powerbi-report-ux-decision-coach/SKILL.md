# Power BI Report UX Decision Coach

Choose visuals from the decision task, not from decoration.

## Visual Routing

- Funnel: conversion and stage loss.
- Waterfall: bridge and variance explanation.
- Matrix: controllable operational exceptions.
- Pareto: defect, supplier, product, or customer concentration.
- Aging: overdue work, backlog, SLA, cash, CAPA.
- Flow/process view: handoff and bottleneck.
- Cohort: churn, retention, repeat defects, renewals.
- Map: geography only when location changes action.

## Output Contract

- Page purpose, audience, primary decision, and next action.
- Visual selection, fields, sort, interactions, drillthroughs, and tooltips.
- Accessibility, density, mobile/responsive notes, and anti-misleading checks.
