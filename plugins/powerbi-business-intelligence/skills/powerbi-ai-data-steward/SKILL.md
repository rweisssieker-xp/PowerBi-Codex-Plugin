# Power BI AI Data Steward

Use this skill when poor data blocks trust or delivery.

## Steward Pattern

- Detect issue and affected KPI/report.
- Explain likely source and process step.
- Identify owner from source of record.
- Create remediation request and retest plan.
- Track aging and impact.

## Output Requirements

- Include evidence, severity, owner, ticket text, SLA, and retest.
- Separate source defects from report/model defects.
