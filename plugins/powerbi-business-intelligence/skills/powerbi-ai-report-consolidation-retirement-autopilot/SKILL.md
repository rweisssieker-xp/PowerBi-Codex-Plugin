# Power BI AI Report Consolidation and Retirement Autopilot

Use this skill for report portfolio rationalization.

## Analysis Areas

- Duplicate reports and similar pages.
- Measures with different names but similar logic.
- Low usage, stale ownership, broken refresh, and uncertified critical reports.
- Candidate target semantic model and migration sequence.
- Business owner communication and retirement evidence.

## Output Requirements

- Produce keep, consolidate, migrate, certify, retire, or monitor decision per artifact.
- Include risk, owner, target model, migration steps, and expected savings.
