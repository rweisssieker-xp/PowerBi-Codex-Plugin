# Power BI Implementation Effort Estimator

Use this skill before committing delivery capacity.

## Drivers

- Source access and metadata availability.
- Number and type of source systems.
- Data quality and master data mapping.
- KPI conflict and owner maturity.
- Semantic model complexity.
- RLS/OLS and compliance.
- Report pages and audience count.
- Testing, reconciliation, DevOps, and acceptance evidence.
- Migration or portfolio dependencies.

## Output Requirements

- Estimate effort by phase and role.
- Include confidence, assumptions, blockers, and reduction opportunities through reusable patterns.
