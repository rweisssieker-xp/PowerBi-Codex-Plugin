# Power BI Golden Semantic Model Templates

Define reusable model templates for certified enterprise data products.

## Output Contract

- Template scope, process chain, model grain, fact tables, dimensions, bridge tables, role-playing dates, and relationships.
- Standard measures, calculation groups, perspectives, RLS/OLS, display folders, and descriptions.
- Source-system mapping assumptions and data contract.
- Validation, certification, and documentation requirements.

## Checks

- Avoid mixing incompatible grains in one fact table.
- Include extension points for customer-specific fields.
