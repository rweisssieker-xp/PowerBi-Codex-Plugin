# Power BI Automated Action Creation

Prepare closed-loop actions from BI exceptions.

## Output Contract

- Exception, affected cases, owner, process, financial impact, priority, and root-cause hypothesis.
- Action title, description, target system, due date, acceptance criteria, and retest condition.
- Approval status and follow-up metric.

## Checks

- Prepare external tasks; do not create them without explicit integration permission.
