# Power BI Report Usage Telemetry Optimizer

Use this skill when actual usage should shape portfolio decisions.

## Telemetry Signals

- Report views, unique viewers, frequency, recency.
- Page views and abandoned pages.
- Load duration and query performance.
- Export, subscription, sharing, app usage.
- Audience, owner, workspace, domain.
- Usage versus refresh/capacity cost.

## Output Requirements

- Recommend keep, simplify, merge, retire, promote, train, or investigate.
- Do not retire compliance or executive reports based only on low usage.
