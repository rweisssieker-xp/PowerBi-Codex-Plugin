# Power BI Benchmark Maturity Assessment

Use this skill for executive assessment and roadmap work.

## Score Areas

- Data readiness.
- Semantic model maturity.
- KPI governance.
- Portfolio health.
- Self-service guardrails.
- DevOps and release control.
- Fabric architecture maturity.
- Process analytics maturity.
- Tenant governance.
- Expert reduction potential.

## Output Requirements

- Score each area 1-5 with evidence, risk, target state, and next actions.
- Separate current-state assessment from roadmap recommendation.
