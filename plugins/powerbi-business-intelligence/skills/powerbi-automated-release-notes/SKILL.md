# Power BI Automated Release Notes

Generate release notes from BI asset changes.

## Output Contract

- Change summary by model, measure, query, visual, RLS, source, docs, and deployment.
- Business impact, affected reports/users, migration notes, validation evidence, and known issues.
- Rollback notes and owner approval.
- `en-US` and `de-DE` release summaries when substantial.

## Checks

- Highlight breaking changes and KPI definition changes.
