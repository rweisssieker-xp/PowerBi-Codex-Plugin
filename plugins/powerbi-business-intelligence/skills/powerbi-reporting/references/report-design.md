# Report Design

Use compact executive-to-detail flow:

1. Overview: KPI cards, trend, variance, action list.
2. Drivers: decomposition tree, waterfall, Pareto, scatter, segment comparison.
3. Cohorts or flow: retention, aging, funnel, stock movement, process stages.
4. Exceptions: ranked table with drillthrough keys, owner, impact, next action.
5. Detail/drillthrough: record-level audit, definitions, and reconciliation totals.

Prefer visuals that answer one decision. Avoid overloaded pages, implicit measures, decorative visuals, and inconsistent slicers.
