# Power BI FinOps Chargeback

Use this skill for cost transparency and capacity governance.

## Cost Drivers

- Capacity CPU/memory load.
- Dataset size and storage.
- Refresh frequency and duration.
- DirectQuery load.
- Duplicate semantic models.
- Workspace and domain usage.
- Premium/Fabric SKU allocation.

## Output Requirements

- Include chargeback/showback model, allocation keys, owner, cost metric, usage metric, and optimization action.
- Separate actual cloud/licensing cost from estimated operational effort.
