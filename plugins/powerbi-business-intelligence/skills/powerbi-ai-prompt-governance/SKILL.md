# Power BI AI Prompt Governance

Use this skill when natural-language BI is exposed to users.

## Policy Areas

- Allowed and prohibited question classes.
- PII, HR, finance, export-controlled, and regulated data handling.
- Role-based prompt access.
- Prompt templates and versioning.
- Audit trail and retention.
- Test set and hallucination checks.

## Output Requirements

- Include prompt policy, examples, blocked cases, owner, and review cadence.
