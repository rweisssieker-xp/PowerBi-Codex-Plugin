# Power BI Reference Architecture Generator

Use this skill when the solution needs a target architecture, not only a report.

## Architecture Inputs

- Source systems and ownership.
- Data volume, latency, refresh, and security requirements.
- Existing data platform: Fabric, Synapse, Snowflake, Databricks, SQL Server, BW, Datasphere, lakehouse.
- Integration style: batch, CDC, API, OData, file, event streaming.
- Governance: domains, workspaces, deployment, RLS, certification, lineage.

## Output Requirements

- Provide recommended architecture, fallback architecture, data flow, semantic model strategy, workspace layout, deployment path, and operational risks.
- Include specific fit for Import, DirectQuery, Direct Lake, dataflows, warehouse, lakehouse, and certified datasets.
