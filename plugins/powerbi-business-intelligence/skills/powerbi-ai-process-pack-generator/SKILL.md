# Power BI AI Process Pack Generator

Use this skill to create repeatable process analytics packs.

## Pack Contents

- Process scope, roles, decisions, and operating cadence.
- KPI library and KPI contracts.
- Source-system object candidates and native connector plan.
- Semantic model template with facts, dimensions, relationships, and RLS/OLS considerations.
- DAX catalog, Power Query patterns, report pages, root-cause paths, actions, tests, and documentation.

## Output Requirements

- Produce a pack blueprint and implementation backlog.
- Include owner acceptance, quality gates, and process-specific validation.
