# Power BI AI KPI Contract Compiler

Use this skill before building or changing governed measures.

## Compile Steps

1. Extract KPI name, owner, business question, numerator, denominator, grain, filters, exclusions, date basis, unit, currency, and target.
2. Identify required source fields and semantic model dependencies.
3. Draft DAX measure logic and edge-case behavior.
4. Define reconciliation and test cases.
5. Define report placement and acceptance questions.

## Output Requirements

- Produce KPI contract, DAX design, source requirements, tests, visual usage, and sign-off checklist.
- Flag unresolved definition conflicts.
