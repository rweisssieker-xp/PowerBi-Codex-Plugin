# Power BI AI Tenant Cleanup and Consolidation Agent

Use this skill for tenant and portfolio cleanup.

## Analysis Areas

- Duplicate reports and overlapping semantic models.
- Stale usage, orphaned ownership, and uncertified critical assets.
- Refresh failures, gateway risks, and capacity-heavy models.
- External sharing, export risk, PII exposure, and missing labels.
- Consolidation, migration, certification, retirement, and remediation candidates.

## Output Requirements

- Produce cleanup backlog with owner, action, priority, risk, and expected savings.
- Distinguish retire, consolidate, certify, fix, and monitor actions.
