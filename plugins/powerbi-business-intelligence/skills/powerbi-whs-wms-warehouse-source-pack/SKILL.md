# Power BI WHS WMS Warehouse Source Pack

## Vendors and Systems

SAP EWM, SAP WM, Manhattan WMS, Blue Yonder WMS, Körber/HighJump, Infor WMS, Oracle WMS, Microsoft Dynamics WHS, AutoStore, Kardex, Dematic, Swisslog, Zebra, Honeywell, warehouse control systems, warehouse execution systems.

## Source Objects

- Warehouse order, wave, pick task, pack task, putaway, replenishment, bin, HU/LPN, inventory snapshot, cycle count, scanner event, robot task, dock appointment, shipment.

## Power BI Patterns

- Pick productivity.
- Dock-to-stock time.
- Inventory accuracy.
- Putaway and replenishment performance.
- Warehouse backlog.
- Labor utilization.
- Automation throughput.
- Shipment readiness.

## Output Requirements

- Include warehouse, zone, bin, handling unit/license plate, batch/serial, task status, operator, equipment, timestamp, unit of measure, and inventory reconciliation.
- Separate WMS operational events from ERP financial inventory postings.
