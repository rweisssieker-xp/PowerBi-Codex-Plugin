# Power BI AI Multi Agent BI Orchestrator

Coordinate specialist BI agents across a delivery lifecycle.

## Output Contract

- Delivery goal, scope, constraints, artifact list, and quality gate.
- Agent roles: source mapper, modeler, DAX tester, M generator, UX reviewer, governance reviewer, release manager, adoption coach.
- Work breakdown, dependencies, handoff artifacts, review gates, and escalation.
- Final integrated delivery pack and evidence trail.

## Checks

- Avoid parallel work on the same artifact without ownership boundaries.
- Require integration review before completion.
