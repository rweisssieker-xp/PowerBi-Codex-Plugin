# Power BI Pattern Marketplace

Use this skill to turn repeated Power BI expert work into reusable patterns.

## Pattern Types

- Domain packs: customer churn, material churn, FiCO, sales, procurement, service, HR, risk.
- Page patterns: executive overview, driver analysis, cohort, flow, exception table, drillthrough.
- KPI patterns: rate, variance, aging, retention, funnel, SLA, working capital.
- DAX patterns: time intelligence, cohort, bridge, ranking, segmentation, dynamic metric.
- RLS patterns: organization, customer, branch, cost center, region, manager hierarchy.
- Validation patterns: source totals, row count, KPI reconciliation, UAT, RLS denied access.

## Output Requirements

- Define pattern name, use case, inputs, outputs, constraints, validation, and examples.
- Mark patterns as draft, approved, deprecated, or blocked.
