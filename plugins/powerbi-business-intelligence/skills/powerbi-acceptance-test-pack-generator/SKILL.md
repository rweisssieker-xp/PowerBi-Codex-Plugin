# Power BI Acceptance Test Pack Generator

Create business acceptance packs for release and certification.

## Output Contract

- Test scope, owner, role, process, report page, KPI, and source reconciliation.
- Test cases with steps, filters, expected values, tolerance, evidence, and pass/fail result.
- RLS, performance, refresh, documentation, and usability acceptance.
- Sign-off matrix and open issue log.

## Checks

- Include case-level source tracing for critical KPIs.
- Separate UAT defects from change requests.
