# Power BI AI Semantic Model Diff and Regression

Use this skill before accepting model changes.

## Diff Areas

- Added, removed, renamed, or type-changed columns.
- Added, removed, renamed, or changed measures.
- Relationship, cardinality, direction, and active-state changes.
- RLS/OLS and perspective changes.
- Report visual impact and broken bindings.
- KPI contract and certification impact.

## Output Requirements

- Include breaking changes, non-breaking changes, affected artifacts, regression tests, and mitigation.
- Mark untested measure changes as risks.
