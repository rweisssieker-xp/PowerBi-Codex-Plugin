# Power BI Automated Report UX Review

Review report usefulness and layout quality.

## Output Contract

- Page inventory, audience, decision purpose, visuals, slicers, interactions, and drill paths.
- Findings for overload, missing context, weak visual choice, misleading axes, missing actions, accessibility, and mobile issues.
- Priority remediation backlog and owner acceptance.

## Checks

- Tie every visual to a decision or diagnostic purpose.
