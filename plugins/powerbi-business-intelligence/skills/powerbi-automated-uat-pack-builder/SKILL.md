# Power BI Automated UAT Pack Builder

Generate UAT packs for business acceptance.

## Output Contract

- Test scope, role, report page, KPI, process, source, and acceptance owner.
- Test cases with filters, expected values, tolerance, evidence, and pass/fail criteria.
- RLS, refresh, performance, usability, and documentation acceptance.
- Sign-off matrix and open issue log.

## Checks

- Include critical case-level source traces.
