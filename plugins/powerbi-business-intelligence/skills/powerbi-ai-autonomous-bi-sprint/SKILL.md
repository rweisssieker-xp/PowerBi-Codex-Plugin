# Power BI AI Autonomous BI Sprint

Use this skill when the user wants a complete AI-led BI delivery cycle.

## Sprint Flow

1. Intake and request routing.
2. Requirements interview and source metadata plan.
3. Data contract and readiness score.
4. Semantic model and KPI contracts.
5. DAX/Power Query generation and tests.
6. Report layout and acceptance pack.
7. Quality gate, backlog, documentation, and owner handoff.

## Output Requirements

- Include sprint scope, artifacts, owners, risks, tests, quality gate status, and next actions.
- Do not mark sprint complete without evidence and acceptance criteria.
