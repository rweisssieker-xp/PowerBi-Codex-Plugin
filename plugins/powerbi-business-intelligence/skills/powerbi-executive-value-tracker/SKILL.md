# Power BI Executive Value Tracker

Use this skill to prove the plugin's value to leadership.

## Value Metrics

- Expert hours avoided.
- Reports retired, merged, or certified.
- Duplicate semantic models eliminated.
- KPI conflicts resolved.
- Refresh failures reduced.
- Delivery cycle time reduced.
- Capacity/storage cost reduced.
- Self-service adoption increased.
- Quality gate pass rate.
- Business owner acceptance time reduced.

## Output Requirements

- Include baseline, target, current value, owner, evidence source, and confidence.
- Separate hard savings from estimated productivity gains.
