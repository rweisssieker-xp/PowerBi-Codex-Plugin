# Power BI APM Observability ITOps Source Pack

## Vendors

Splunk, Datadog, Dynatrace, New Relic, Azure Monitor, Grafana, Prometheus, Elastic, AppDynamics.

## Source Objects

- Metric, log event, trace, alert, incident, service, host, dependency, dashboard, SLO, error, latency, capacity event.

## Power BI Patterns

- Data platform health.
- Power BI/Fabric operations.
- SLA/SLO tracking.
- Incident trend.
- Cost/performance correlation.
- Refresh failure root cause.

## Output Requirements

- Include timestamp grain, retention, metric namespace, service ownership, alert severity, and incident linkage.
