# FAQ

## en-US

## What is this repository?

It is an AI-native Power BI Expert-Replacement Factory plugin for Codex. It turns process questions into governed Power BI delivery artifacts.

## Does it replace business accountability?

No. It replaces repeatable expert work. Process Owners still approve KPI meaning, source ownership, and action accountability.

## Are generated Power BI artifacts production-ready automatically?

No. Production-grade artifacts require quality gates, source validation, KPI contracts, security review, and Power BI validation evidence.

## Are docs bilingual?

Yes. Substantial documentation should include `en-US` and `de-DE`.
