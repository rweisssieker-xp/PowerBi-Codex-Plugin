# Support

Use this repository for Power BI Expert-Replacement Factory issues that are reproducible, actionable, and safe to discuss publicly.

Use GitHub issues for:

- Bugs in documentation, skills, scripts, schemas, templates, or generated sample artifacts.
- Feature requests for new skills, process packs, validators, or governance patterns.
- Process-pack requests such as Lead2Order, Order2Cash, Procure2Pay, or Record2Report.
- Documentation gaps or corrections.

Do not use public GitHub issues for:

- Security vulnerabilities.
- Secrets, credentials, tokens, tenant IDs, gateway details, or API keys.
- Customer-sensitive data extracts.
- Confidential screenshots or Power BI files.

For security-sensitive reports, follow [SECURITY.md](SECURITY.md).
