# Support

## en-US

Use this repository for Power BI Expert-Replacement Factory issues that are reproducible, actionable, and safe to discuss publicly.

Use GitHub issues for:

- Bugs in documentation, skills, scripts, schemas, templates, or generated sample artifacts.
- Feature requests for new skills, process packs, validators, or governance patterns.
- Process-pack requests such as Lead2Order, Order2Cash, Procure2Pay, or Record2Report.
- Documentation gaps and bilingual `en-US` / `de-DE` corrections.

Do not use public GitHub issues for:

- Security vulnerabilities.
- Secrets, credentials, tokens, tenant IDs, gateway details, or API keys.
- Customer-sensitive data extracts.
- Confidential screenshots or Power BI files.

For security-sensitive reports, follow [SECURITY.md](SECURITY.md).

## de-DE

Nutze dieses Repository fuer Power BI Expert-Replacement Factory Themen, die reproduzierbar, konkret und oeffentlich sicher diskutierbar sind.

Nutze GitHub Issues fuer:

- Fehler in Dokumentation, Skills, Scripts, Schemas, Templates oder generierten Beispielartefakten.
- Feature Requests fuer neue Skills, Process Packs, Validatoren oder Governance Patterns.
- Process-Pack-Anfragen wie Lead2Order, Order2Cash, Procure2Pay oder Record2Report.
- Dokumentationsluecken und Korrekturen fuer `en-US` / `de-DE`.

Nutze oeffentliche GitHub Issues nicht fuer:

- Sicherheitsluecken.
- Secrets, Credentials, Tokens, Tenant IDs, Gateway Details oder API Keys.
- Kundensensitive Datenextrakte.
- Vertrauliche Screenshots oder Power-BI-Dateien.

Fuer sicherheitssensitive Meldungen gilt [SECURITY.md](SECURITY.md).
