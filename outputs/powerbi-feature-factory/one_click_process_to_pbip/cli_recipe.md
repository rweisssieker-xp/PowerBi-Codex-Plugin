# One-Click Process-to-PBIP CLI Recipe

Turn a selected process pack into a PBIP/TMDL/PBIR generation target.

## Commands

```powershell
python scripts\build_industry_process_packs.py
```
```powershell
python scripts\powerbi_expert_factory.py feature-plan --process <processId> --out <file>
```

## Acceptance Checks

- process pack exists
- model spec exists
- report page specs exist
