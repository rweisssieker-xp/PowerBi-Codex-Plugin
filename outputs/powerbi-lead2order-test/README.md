# Lead2Order Process Owner Analytics Test Pack

## en-US

This test pack translates the role of a Lead2Order Process Owner into a complete Power BI analytics scope. It defines the relevant evaluations, KPIs, semantic model, report pages, DAX measure catalog, action cockpit, and quality gates needed to manage the process from lead intake to booked order.

## de-DE

Dieses Testpaket uebersetzt die Rolle eines Lead2Order Process Owners in einen vollstaendigen Power-BI-Analytics-Scope. Es definiert die passenden Auswertungen, KPIs, das semantische Modell, Reportseiten, DAX-Measure-Katalog, Action Cockpit und Quality Gates fuer die Steuerung vom Lead Intake bis zum gebuchten Auftrag.

## Files

- `LEAD2ORDER_PROCESS_OWNER_ANALYTICS.md`: bilingual process-owner analysis specification.
- `lead2order_process_pack.json`: machine-readable process pack.
- `kpi_contracts.json`: KPI contract candidates for owner approval.
- `dax_measures.dax`: DAX measure starter catalog.
- `semantic_model.md`: semantic model and relationship design.
- `quality_gate_plan.json`: release and validation gates.
